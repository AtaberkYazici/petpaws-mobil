# petpaws

Bu proje hayvan sahiplendirme sürecinin kolaylaştırılması için flutter kullanılarak yapılmıştır. PetPaws api projesi çalıştırılıp mssql ile bağlantısı yapıldıktan sonra kullanılması gerekmektedir.
